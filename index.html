Membuat website responsive menggunakan HTML & CSS.

Desain Web Responsif adalah tentang menggunakan HTML dan CSS untuk secara otomatis mengubah ukuran, menyembunyikan, mengecilkan, atau memperbesar, sebuah situs web, agar terlihat bagus di semua perangkat (desktop, tablet, dan ponsel. Ini akan mengatur area pandang halaman Anda, yang akan berikan petunjuk browser tentang cara mengontrol dimensi dan penskalaan halaman.
